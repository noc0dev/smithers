# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-30

### Added

- Smithers skill for autonomous parallel implementation with git worktrees
- Pluggable task sources: natural language, GitHub issues, beads integration
- smithers-worker agent for isolated worktree implementation
- Automatic CI polling and review handling loop
- Support for CodeRabbit, Codex, and roborev code review integration
